# MandarinbankApi

Простое API для mandarinbank.com

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mandarinbank_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mandarinbank_api
